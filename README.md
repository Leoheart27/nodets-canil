# nodets-canil

Node + Typescript

### Start project
`npm init`

### Global dependencies:
`npm install -g nodemon`
`npm install -g typescript`
`npm install -g ts-node`

### Types dependencies:
`npm install @types/express`
`npm install @types/mustache-express`
`npm install @types/node`

### Project run
`npm run start-dev`
